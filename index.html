Cómo está hecho (gratis)

  1. Clona este repo / archivo en tu GitHub.
  2. Activa GitHub Pages en Settings → Pages → deploy from main.
  3. Edita el array DATA.projects y DATA.cv para cargar tu contenido.
  4. Arrastra tus imágenes a /assets (crea la carpeta) y enlázalas en los objetos.
  5. Personaliza colores en :root y textos de secciones.
